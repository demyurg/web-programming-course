# LR9 Standalone Slides

Это standalone версия слайдов LR9 (Database & Business Logic - Scoring Algorithm) со всеми встроенными зависимостями.

## Использование

1. Откройте файл `slides.html` в браузере
2. Используйте стрелки или пробел для навигации между слайдами
3. Нажмите `ESC` для режима обзора
4. Нажмите `S` для открытия speaker notes (если доступны)

## Что встроено

- Reveal.js 4.5.0 - фреймворк для создания презентаций
- CSS стили (reset.css, reveal.css, black.css, monokai.css)
- JavaScript плагины (highlight, markdown, notes)

## Преимущества

- Полностью независим от интернета
- Работает offline
- Все зависимости находятся в одном файле
- Быстрая загрузка

## Размер файла

- ~1.1 MB (со сжатием reveal.js и всех ресурсов)

## Горячие клавиши

- **Стрелки** - навигация по слайдам
- **Пробел** - следующий слайд
- **F** - полноэкранный режим
- **ESC** - режим обзора (overview)
- **B** - черный экран (pause)
- **S** - speaker notes
